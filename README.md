# heying
合影是一个以豆瓣api为数据来源，实现实时更新上映电影，邀约合影的平台，采用flask框架，后台数据库mysql，orm数据模型，jinja引擎渲染模板 主要模块有：正在热映、即将上映、合影心愿、我的合影、用户设置
